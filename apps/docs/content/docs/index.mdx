Welcome to the Prood docs. This monorepo ships a Next.js 16 storefront, standalone checkout, admin dashboard, and shared packages for payments, storage, and catalog data.

## What is Prood?

Prood is a **commerce-agnostic** toolkit: swap adapters, payment providers, and storage backends without rewriting your storefront.
